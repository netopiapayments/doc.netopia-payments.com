# Errors & Statuses

## Error codes

-   **100** : Set authenticationToken & ntpID to session
-   **56** : duplicated Order ID
-   **99** : There is another order with a different price
-   **19** : Expire Card Error
-   **20** : Founduri Error
-   **21** : CVV Error
-   **22** : CVV Error
-   **34** : Card Tranzactie nepermisa Error
-   **0** : Card has no 3DS

## Status codes

-   **3** : is paid
-   **5** : is confirmed
-   **12** : Invalid Account
-   **15** : need authorize
